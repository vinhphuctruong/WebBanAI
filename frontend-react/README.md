# Frontend Next.js

## Run
```bash
cp .env.example .env
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Environment
- `NEXT_PUBLIC_API_BASE`: API endpoint cho browser/client components.
- `API_BASE_INTERNAL`: API endpoint cho server-side rendering trong Next.
- `NEXT_PUBLIC_SITE_URL`: domain dùng cho metadata/canonical.
